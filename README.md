# 📱 Ok Tally - Digital Ledger & Expense Tracker

*Ok Tally* is a lightweight, intuitive, and user-friendly Android application built using the Flutter framework and Dart programming language. It is designed to help users seamlessly manage their daily transactions, track expenses, and maintain a digital ledger with absolute ease.

---

## ✨ Key Features

* *🎨 Clean & Minimal UI:* A modern, clutter-free user interface designed for smooth navigation and effortless daily bookkeeping.
* *🖼️ Custom Branding:* Integrated with a unique, high-quality custom logo for a professional look and feel.
* *🔝 Real-Time Smart Sorting:* Automatically sorts entries in real-time, ensuring that your most recent transactions always appear right at the top of the list.
* *⚡ Highly Optimized:* Fast, responsive, and lightweight, making it compatible with almost any Android device without lagging.

---

## 🛠️ Built With

* *Framework:* Flutter (Cross-platform UI Toolkit) 🚀
* *Language:* Dart 🎯
* *Platform Support:* Android 🤖

---

## 📲 How to Download & Install

To test or use the app directly on your Android device:
1. Navigate to the *Releases* section on the right side of this GitHub page.
2. Download the latest app-release.apk file.
3. Open the downloaded file on your phone and complete the installation (enable "Install from Unknown Sources" if prompted).
